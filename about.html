About F1 Fan Hub

Our Mission

F1 Fan Hub helps fans organize race weekends, manage watch parties, and track personal plans with task analytics.

The app demonstrates modern web development using Bootstrap, vanilla JavaScript DOM manipulation, localStorage data persistence, API integration, and interactive dashboards.

FAQ

F1 Fan Hub is a task manager for fans to plan race weekends, watch parties, meetups and reminders.

Tasks are saved in localStorage, so they remain available after refresh without a server.

It visualizes completed vs pending tasks and shows a priority breakdown (High/Medium/Low).
